# MISP Modules

MISP modules are self-contained modules that provide expansion and other services for MISP.
They are written in Python 3 and follow a simple API interface.
The goal is to extend MISP functionality without modifying core components.
The API is exposed through a REST interface that is independent of the main MISP installation and configuration.

Support for MISP modules is included in MISP starting with version 2.4.28.
